Reusable Data Pipeline

This project is a modular, scriptable data pipeline that enables reading and writing to various services like Google Drive, Sheets, Docs, Calendar, Forms, Dropbox, Slack, and more.

Purpose

This pipeline is designed for developers and teams who want to automate data workflows across cloud services using a unified, configurable interface.

Features

  • Modular Fetchers and Writers
  • Support for multiple Google APIs (Drive, Sheets, Docs, etc.)
  • Dropbox, Slack, Airtable, Salesforce integration
  • YAML-based configuration
  • Supports CLI usage and embeddable as a Python package

Disclaimer

This is a backend tool. It does not have a public-facing UI. Users configure their own credentials and services locally. The tool does not access user data beyond what is explicitly authorized during OAuth.
